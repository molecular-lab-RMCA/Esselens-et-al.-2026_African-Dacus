# Genome-scale phylogeny of African Dacus

This repository contains alignments, phylogenetic trees, chronogram files, and metadata associated with:

Esselens et al. Genome-scale phylogeny of African Dacus (Diptera: Tephritidae) refines evolutionary hypotheses and reveals extensive morphological homoplasy.

## Contents

- Nuclear alignment: 500 orthogroups
- Mitochondrial alignment: 13 mitochondrial protein-coding genes
- Chronogram dataset: 20 conserved marker genes
- Maximum-likelihood trees
- MCMCTree chronogram output
- Specimen and host-use metadata

## Data summary

The dataset includes 179 African Dacus specimens representing 51 species. Nuclear phylogenetic inference used 500 orthogroups. The mitochondrial dataset included 13 protein-coding genes. Divergence-time estimation used 20 conserved marker genes.
