# Metadata

This folder contains specimen-level and trait metadata associated with the African *Dacus* phylogenomics dataset.

## Files

`metadata.csv`  
  - Specimen-level information for the 179 African *Dacus* specimens included in the study.
  - Host plant family per species

## Column descriptions


| Column | Description |
|---|---|
| `species` | Scientific name of the specimen. |
| `country` | Country where the specimen was collected. |
| `locality` | Collection locality. |
| `latitude` | Latitude of the collection locality. |
| `longitude` | Longitude of the collection locality. |
| `collection_year` | Year in which the specimen was collected. |
| `voucher_id` | Voucher identifier. |
| `dna_voucher_reused` | Indicates whether the DNA voucher was reused from Virgilio et al. (2009). |
| `host_plant_family` | Recorded host plant family, when known. |
