# OMA groups

This folder contains the OMA groups retrieved from the OMA Browser and used for the chronogram dataset in the African *Dacus* phylogenomics study.

## Description

The files in this folder correspond to the conserved orthologous marker genes selected for divergence-time estimation. These marker genes were retrieved from the OMA Browser using the “Export marker genes” function and were chosen based on representation across the reference taxa used for the chronogram analysis.

The resulting OMA groups were used to construct the reduced alignment for Bayesian divergence-time estimation with MCMCTree.

## Files

| File                        | Description                                                                       |
| --------------------------- | --------------------------------------------------------------------------------- |
| `OG*.fna`                   | Individual OMA group alignments or sequence files retrieved from the OMA Browser. |

## Use in this study

These OMA groups were used to build the reduced 20-locus alignment for divergence-time estimation.
The resulting alignment was analysed with MCMCTree under a relaxed molecular-clock framework using fossil-based calibrations.
